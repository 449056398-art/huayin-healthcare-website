# Huayin Healthcare Overseas Website

## Start locally

Double-click `启动网站.cmd` in this folder. It starts the local website and opens:

`http://127.0.0.1:5173`

Do not open `index.html` directly in a browser. This is a React + Vite application and needs the local server to run.

## Build for deployment

```bash
pnpm build
```
